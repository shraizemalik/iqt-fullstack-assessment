# Frontend

React + Vite client for the Task Management System.

## Environment Variables

- `VITE_API_BASE_URL`: Laravel API base URL, for example `http://127.0.0.1:8000/api/v1`
- `VITE_REPOSITORY_URL`: repository link shown in the top navigation

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```
